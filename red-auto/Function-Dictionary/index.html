The Red stack consists of two main layers, the high-level Red language and a lower-level DSL called Red/System, meant for system programming. The high-level language is very similar with Rebol2 (roughly 95% source-compatible), so the Rebol documentation can be used for learning Red until we get our own one. The reference documentation for Red/System is available below.
